Article: Rhetorical Shifts in European Union Foreign Policy: The Impact of Emotional Catalysts Following the Full-Scale Invasion of Ukraine
This repository contains all the code used in the analysis of the above-titled academic article. This code was created by Sasha Nielsen as part of the PhD project at the Department of Global Studies, Aarhus University, Denmark.
Included in this repository is code used to scrape the results from the EU Foreign Policy statements and press releases website as well as complete a fine-grained emotion-based analysis using the EmoRoBERTa model.
